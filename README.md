# محضر · Mahdar

> **من المحضر إلى الإنجاز** — نظام إدارة المجالس والاجتماعات: المحاضر، القرارات، التكليفات، والمتابعة التنفيذية.

## التقنيات
Next.js 16 · React 19 · TypeScript · Prisma · Tailwind v4 · jose (JWT) · عربي RTL

قاعدة البيانات: **SQLite** محليًا · **PostgreSQL** للإنتاج (يتحوّل تلقائيًا عبر `scripts/prepare-local-prisma.mjs`).

## التشغيل المحلي
```bash
npm install
npm run db:local:prepare   # توليد العميل + إنشاء قاعدة SQLite
npm run db:seed            # بيانات تجريبية
npm run dev                # http://localhost:3000
```

## حسابات تجريبية (كلمة المرور: `12345678`)
| المستخدم | الدور |
|---|---|
| `amin` | أمين السر |
| `rais` | رئيس المجلس |
| `edu` | مسؤول لجنة التعليم |
| `fin` | مسؤول اللجنة المالية |
| `member1` / `member2` | عضو |

## المزايا المنجزة
- ✅ **المصادقة والصلاحيات** — دخول باسم مستخدم/كلمة مرور، 4 أدوار (أمين السر · رئيس · مسؤول قسم · عضو) بحماية حقيقية.
- ✅ **المجالس واللجان** — هيكل شجري، أعضاء (دائم/ضيف)، أقسام بمسؤولين، دورية اجتماعات.
- ✅ **الاجتماعات** — إنشاء، جدول أعمال، تسجيل الحضور والغياب، حالات الاجتماع.
- ✅ **محضر المحاضر** — بنود بأنواعها (نقاش/قرار/تكليف/متابعة/تكلفة/ملاحظة/تصويت)، وتحويلها مباشرةً إلى **تكليفات وتكاليف**.
- ✅ **الاعتماد والإقفال** — مسودة → مراجعة → معتمد → مقفل، ملحق رسمي، سجل تدقيق (Audit Log).
- ✅ **المتابعة التنفيذية** — لوحة تكليفات بفلاتر، روزنامة استحقاقات، عرض شجري، تحديث الحالة والملاحظات.
- ✅ **الأرشيف والتقارير** — بحث وفلاتر، تصدير المحضر PDF، تقارير الأقسام بضغطة زر + تصدير Excel.
- ✅ **التذكيرات والإشعارات** — تذكير باجتماعات المجلس، جرس إشعارات داخلي (بنية جاهزة لـ WhatsApp/Email).

> **5 مراحل من 6 مكتملة** — كل وحدات الـ MVP الـ12 جاهزة. البناء الإنتاجي يمرّ بلا أخطاء (`npm run build`).
